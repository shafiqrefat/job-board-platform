## Getting Started

First, run the development server:

```bash
npm run dev

🚀 Features

🔍 Search Functionality
- Easily search for job listings based on jobs,companies and location.
🧮 Sorting Options
- Sort jobs by types, categories and location.
🎯 Filtering Capabilities
- Filter listings by job type, location, and categories.
📝 User Registration & Login
- Secure signup and login functionality for job poster.
📤 Job Posting
- Authenticated users can post new job opportunities.
🛡️ Protected Routes
- Job posting routes are protected and accessible only to logged-in users.
🔁 Dynamic Routing
- Each job has a dedicated, dynamic route for detailed views like description,requirement and company info.
